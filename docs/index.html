A Flat Image Is a Dead End for Editing

When design files are lost, only a flat image remains. You cannot move elements, change colors, fix text, or reflow layouts.

What ReDesign Does

ReDesign recovers a fully editable design hierarchy from a single image.

A vision-language controller grows a layer tree breadth-first; a verifier checks every step. The output: semantic text layers with real fonts, vector shapes with fills, groups with z-order, and isolated elements that survive real edits without disturbing the rest of the design.

The Agentic Approach: Grow the Tree

Starting from the whole image as root, a VLM controller expands breadth-first. At each node: choose one tool-backed action; a verifier checks validity. Repeat until every leaf is atomic and editable.

Why Structure Beats Pixel Regeneration

Generative editors (Nano Banana 2) edit pixel-space: instructions leak, layouts blur, precision fails. ReDesign exposes discrete layers with explicit position and rotation; edits execute exactly.
